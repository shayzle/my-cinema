# my-cinema

An administration dashboard for managing a cinema, built as part of my studies
at Web@cadémie by Epitech.

---

## About the Project

My Cinema is a solo school project built at Web@cadémie by Epitech. It is an
administration dashboard for managing a cinema, designed to practice and understand
PHP Object-Oriented Programming, MVC Architecture, CRUD Logic, Database Relationships,
and PDO for database security. This project was built for educational purposes only.

## Prerequisites

- PHP 8+
- MySQL
- Local server (MAMP, XAMPP, or WAMP)

---

## Installation

1. Clone the repository and place the project in your server root:

    htdocs/My_Cinema_Webacademie/

2. Start MySQL and import the database:

    Database/script.sql

3. Configure the database connection file:

    Back-End/config/database.php

4. Use the following default credentials:

    - Host: 127.0.0.1
    - Database: my_cinema
    - User: root
    - Password: (empty)

5. Start your local server and open your browser at:

    http://127.0.0.1:8888/my-cinema/Front-End/index.html

---

## Architecture

- Backend: PHP with MVC pattern and PDO
- Frontend: HTML, CSS, JavaScript
- Frontend and backend communicate via a JSON API using fetch
